# ☕ Eta Cheirin Bão

Uma página web desenvolvida com foco em apresentar uma proposta visual agradável e acolhedora para uma cafeteria.

---

## 📌 Sobre o projeto

O **Eta Cheirin Bão** é um projeto front-end criado utilizando **HTML, CSS e JavaScript**, com o objetivo de simular a presença online de uma cafeteria.

A aplicação destaca produtos, identidade visual e organização de conteúdo, proporcionando uma experiência simples, bonita e intuitiva para o usuário.

---

## 🖥️ Demonstração

🔗 Acesse o projeto:  
https://eta-cheirin-bao.vercel.app/
---

## 🎯 Objetivos

- Criar uma interface atrativa para uma cafeteria fictícia  
- Praticar estruturação de páginas com HTML  
- Desenvolver estilização moderna e responsiva com CSS  
- Aplicar interatividade básica com JavaScript  
- Evoluir projetos voltados para portfólio  

---

## ⚙️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 🧠 Funcionalidades

- Layout responsivo
- Menu Hamburguer com links de navegação
- Seções organizadas
- Carrossel para imagens
- Design visual voltado para o segmento de cafeteria  
- Navegação simples e intuitiva  

---

## 🎓 Aprendizados

Durante o desenvolvimento deste projeto, foi possível:

- Melhorar a organização de estrutura HTML semântico
- Aprimorar técnicas de estilização com CSS  
- Trabalhar responsividade para diferentes telas  
- Pensar na experiência do usuário (UI/UX)  
- Criar um projeto mais alinhado com aplicações reais
